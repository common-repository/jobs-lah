This plugin provides a shortcode to embed the list of jobs, created by you on JobsLah.com, on your Wordpress site. JobsLah.com allows you to manage the lifecycle of the applicant tracking process. This requires a free employer’s account on JobsLah.com.

== Description ==

https://www.youtube.com/watch?v=SfRGdSPnC_k

**Simplify**
**Organize**
**Better Insight**

The fastest and simplest way to get job board functionality on your Wordpress Site by leveraging the power of JobsLah.com.

This plugin requires a free employers account on JobsLah.com, it replaces the need for cumbersome and complex job board plugins. The plugin allows embedding of jobs, via shortcode, that you have created on the Jobs Lah platform. Alternatively, you can just link directly to your Employer’s landing page on JobsLah.com (see screen shot for example).

The power driving driving your listings is the Jobs Lah platform, a cloud based applicant tracking system for businesses that want to keep track and organize their recruiting efforts without the hassle of setup, configuration, and maintenance of the job board functions. 

Jobs lah also provides analytics to help employers optimize their recruiting strategy over time.

The plugin will gain additional features as time goes on, allowing more customization and options. We welcome feedback from you to help us prioritize our feature list.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress, you are now ready to use the shortcode [jobslah employer=“X”]. Replace X with your Jobs Lah employer number. If you have not registered continue with step 3.
1. Create an employer’s account on https://www.JobsLah.com/employers
1. Fill out your profile, left side menu Profile.
1. Create one or more jobs, left side menu Jobs, click on “Create Job” button.
1. Left side menu, click on Jobs, then click on the Embed tab. You will see instructions for Wordpress, copy the short code (it contains your employer number).
1. Paste this shortcode into the desired page on your Wordpress site.


== Frequently Asked Questions ==

= What is Jobs Lah? =

The best way to think of Jobs Lah is as a cloud based applicant tracking system for small to medium sized businesses that hire often. It provides the functions to create and post jobs on your website and also post individual jobs to Facebook and Twitter. It collects analytics data so you can see where the traffic and views are coming from to better allocation your efforts.

= Is it free? =

Yes, all features will be free while in Beta. There will always be a free tier that allows jobs posting and management. In the future we may introduce premium features that allow deeper insight and integration with recruiting activities.

== Will you add feature X? ==

YES! We have a long backlog of features we want to add. It would help us greatly if you could provide feedback and comments on what you would like to see and what works and doesn’t work so that we can build the best cloud based jobs system for SME businesses.
